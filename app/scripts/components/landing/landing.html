This instance of the Labeling System (LS) contains controlled vocabularies and concepts of mainzed's partner institutes. The main goal is to publish knowledge as keywords to the public. Moreover, it creates interoperabilty between distributed data repositories. The LS is based on Linked Open Data techniques, in particular a limited usage of SKOS.

The Labeling System is a web-application for building up your own special controlled vocabularies. Vocabularies consist of concepts described by a label and translations. In addition, concepts can be linked to other concepts in the Linked Open Data Cloud. These links can be hierarchical, associative or mapping. Moreover, the app provides a convenient way to search through controlled vocabularies (reference thesauri) in the Linked Open Data Cloud. Once a concept is released, it can be publicly accessed via a Uniform Resource Identifier (URI).

The public version of the Labeling System offers the possibility to view and download released vocabularies.

The editor mode is only available for registered users. Please contact us via email if you want to participate.
